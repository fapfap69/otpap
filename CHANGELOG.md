# Changelog

## v2.0.0-draft

Initial public draft of OTPAP v2.

### Added

- Draft RFC and supporting documentation
- Formal token format and protocol flow
- Replay protection and state machine guidance
- JSON Schemas for token, request, and response objects
- PHP reference implementation
- Multi-language example integrations
- Test vectors for valid and invalid requests
- GitHub Actions workflow for schema validation and PHP tests

### Notes

- The protocol is intentionally distinct from JWT.
- The draft prioritizes server-side request binding and one-time consumption.
